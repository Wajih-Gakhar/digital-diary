# 📓 Digital Diary

A desktop digital diary application built in **C++** with **Qt**, featuring real-time entry, calendar-based browsing, and category-based search.

## 🚀 Features

### 📝 Real-Time Diary Entry
- Write and save diary entries instantly
- Edit and delete existing entries

### 🔍 Search & Filter
- Search entries by keyword
- Combine both for precise results

### 📅 Calendar-Based Browsing
- Browse entries by date using an interactive calendar
- Jump to any date to view entries

### 📁 Category-Based Browsing
- Organize entries by category
- Search and filter entries by category

### 💾 Persistent File-Based Storage
- All entries saved across sessions via file handling

## ⚙️ How to Run

### Prerequisites
- Visual Studio 2022 or later
- Qt Visual Studio Tools extension installed
- Qt 6.x installed

### Steps
1. Clone the repository
```bash
   git clone https://github.com/Wajih-Gakhar/digital-diary.git
```
2. Open `DigitalDiary.slnx` in **Visual Studio**
3. Make sure the **Qt Visual Studio Tools** extension is installed
4. Build & Run ▶️
